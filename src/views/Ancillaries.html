显示或隐藏内容
  查看代码

		
.show 和 .hidden 类可以强制任意元素显示或隐藏(对于屏幕阅读器也能起效)。这些类通过 !important 来避免 CSS 样式优先级问题
另外，.invisible 类可以被用来仅仅影响元素的可见性，也就是说，元素的 display 属性不被改变，并且这个元素仍然能够影响文档流的排布。

		

			

				复制代码
				

			

			

				
				
// Classes
				.show {
				  display: block !important;
				}
				.hidden {
				  display: none !important;
				}
				.invisible {
				  visibility: hidden;
				}
